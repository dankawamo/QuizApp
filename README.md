# QuizApp
画像付きの4拓クイズを作成するアプリです

## 概要
PythonおよびGeminiを用いて作成した、画像付きの4択クイズアプリです。  
問題ごとに画像を表示し、選択肢の中から答えを一つ選べるようになっています。

## 作成目的
言語学習の一環として、現在家庭教師として指導している生徒に対し、問題を出題できるツールとして本アプリを開発しました。
プログラミング未経験の状態から、AIを活用しながら本アプリを開発しました。

## 使用技術
- Python
- Gemini

## 主な機能
- 画像付きクイズ表示
- 問題の管理・作成・編集機能

## 工夫した点
- 作成した問題をボタン操作でクイズ画面に反映できるように設計し、出題準備を簡単に行える仕様としました。
- 問題の出題順をドラッグ＆ドロップで自由に変更できるようにし、直感的に操作できるUIを実現しました。

## 実行方法
以下のコマンドを実行することでアプリを起動できます。

```bash
python app.py
